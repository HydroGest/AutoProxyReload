# AutoProxyReload

AutoProxyReload 是一款专为 Windows 用户设计的极简代理辅助工具。它能自动探测并接管局域网内手机代理服务器的连接，让电脑无需额外安装复杂的代理客户端即可通过手机的网络中转访问互联网。

## 为什么你需要 AutoProxyReload

在公共电脑或他人电脑上配置代理软件不仅繁琐，还容易留下配置记录或导致环境冲突。AutoProxyReload 的核心价值在于即插即用：

* **零配置环境**：电脑端无需安装任何代理客户端，无需导入订阅，更无需维护节点。所有代理节点管理都在手机端完成。
* **智能自动重连**：当手机开启 VPN 共享时，电脑会自动感知并接管网络；手机关闭 VPN 后，程序会自动清理代理设置，确保电脑恢复正常网络，避免出现网络黑洞。
* **静默无痕**：程序以极小的内存占用后台运行。它只负责切换 Windows 系统代理开关，关闭程序时会自动清除所有修改，保证电脑环境洁净如初。
* **即插即用**：它是你手机的“无线外设”，手机开梯子，电脑自动上网，无需任何人工干预。

## 手机端操作指南

要使电脑能够通过手机连接代理，需要确保手机端代理软件正确配置了共享通路：

1. 打开手机上的代理软件（如 Clash、V2Ray 等）。
2. 在设置中找到 **“允许局域网连接” (Allow LAN / LAN Mode)** 并将其开启。
3. 确保手机和电脑连接在同一个 Wi-Fi 网络下。
4. 如果你修改了代理端口，请确保手机端开启的端口与本工具内设置的端口保持一致（默认为 7890）。

## 如何启动与管理

1. **运行程序**：直接双击下载包中的 `AutoProxyReload.exe` 即可启动。程序图标将出现在右下角系统托盘。
2. **右键管理**：右键点击托盘图标，你可以查看当前连接状态、重新定位手机、手动修改端口或退出程序。
3. **开机自启动设置**：
* 按下键盘上的 `Win + R` 组合键。
* 输入 `shell:startup` 并按下回车，这会打开 Windows 的启动文件夹。
* 将 `AutoProxyReload.exe` 的快捷方式复制并粘贴到这个文件夹中。
* 此后每次开机，Windows 都会自动启动该工具。



## 功能特性一览

* **全自动探测**：内置并发扫描算法，自动寻找局域网内开启了代理功能的设备 IP，无需用户手动输入。
* **状态实时同步**：托盘图标颜色动态指示连接情况（绿色为正常，黄色为重连中，红色为断开），右键菜单顶部直接显示当前连接状态。
* **安全关闭机制**：软件在退出或检测到 Windows 关机信号时，会自动关闭系统代理并清理注册表残留，防止断网。
* **深色模式支持**：内置现代化的深色 UI 主题，适配 Windows 现代界面视觉体验。
